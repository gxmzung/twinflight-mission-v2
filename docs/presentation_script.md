# 발표용 설명 문장

저희는 ROS2와 PX4의 역할을 단순히 조사하는 데 그치지 않고, 시설 점검용 자율비행 임무 시나리오를 코드 구조로 구성해 보았습니다.

이 프로젝트에서는 `mission.yaml` 파일에 점검 지점을 waypoint로 정의합니다. ROS2 노드는 이 waypoint를 순서대로 읽고 PX4에 Offboard Control 명령과 목표 위치를 전송합니다. PX4는 해당 명령을 받아 자세 제어, 위치 제어, 고도 유지 등 실제 비행 안정화 기능을 수행합니다.

이를 통해 자율비행 드론이 단순한 하드웨어가 아니라, 하드웨어, 비행제어 소프트웨어, 상위 임무 판단 소프트웨어가 연결된 시스템이라는 점을 확인할 수 있습니다.
